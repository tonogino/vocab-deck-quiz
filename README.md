# Sophia Vocabulary Galgame V2

第二版功能：

- 开始界面
- 多存档
- 存档独立保存好感度、正确率、当前单词、已完成事件
- 词库全局共享
- 每个存档可以选择多个词库学习
- 词库管理界面
- 好感度事件界面
- 事件可以中途退出
- 完成事件会记录在当前存档

## 使用方法

直接打开 `index.html`。

## 图片位置

把索菲亚立绘放进：

`characters/sophia/images/`

需要这些文件名：

- normal.png
- happy.png
- sad.png
- shy.png

如果暂时只有一张图，可以复制四份分别改名。

## 重要说明

数据保存在浏览器 localStorage 里。换浏览器、清缓存、换设备都会丢失。
后续可以加入“导出存档”和“导入存档”功能。
